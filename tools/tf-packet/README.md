# @trustforge-protocol/packet

TrustForge cryptographic packet constructor. Build, serialize, and verify secure, tamper-proof packets for the protocol.

## Install

```sh
bun add @trustforge-protocol/packet
# or
npm install @trustforge-protocol/packet
```

## Usage

```sh
bun run src/cli.ts --help   # build / inspect .tfpkt packets
```
