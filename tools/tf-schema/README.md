# @trustforge-protocol/schema

TrustForge schema definition and validation engine. Define strict, verifiable types for capabilities, evidence, and actions.

## Install

```sh
bun add @trustforge-protocol/schema
# or
npm install @trustforge-protocol/schema
```

## Usage

```sh
bun run src/cli.ts validate-all   # validate every fixture against the schemas
```
