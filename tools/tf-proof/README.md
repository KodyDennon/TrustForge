# @trustforge-protocol/proof

TrustForge cryptographic proof generator. Mint verifiable credentials and zero-knowledge proofs for trustless authorization.

## Install

```sh
bun add @trustforge-protocol/proof
# or
npm install @trustforge-protocol/proof
```

## Usage

```sh
bun run src/cli.ts --help   # mint and verify proof bundles
```
