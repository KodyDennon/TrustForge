# macOS LaunchDaemon for tf-daemon

`com.trustforge.daemon.plist` is a system-scope launchd job that starts `/usr/local/bin/tf-daemon` as the `_trustforge` service user, logs to `/Library/Logs/TrustForge/`, and registers a launchd-managed Unix socket at `/var/run/trustforge/decide.sock` so the daemon is socket-activated on first request. To install: create the `_trustforge` user/group (e.g. via `dscl` or your installer pkg), copy this file to `/Library/LaunchDaemons/com.trustforge.daemon.plist` with `root:wheel` ownership and `0644` mode, then load with `sudo launchctl bootstrap system /Library/LaunchDaemons/com.trustforge.daemon.plist` (or `sudo launchctl load -w …` on older macOS). Check status with `sudo launchctl print system/com.trustforge.daemon`. **Draft / experimental** per `SECURITY.md`; sign and notarize the binary before shipping outside development hosts.
