# @trustforge-protocol/cli

Official Command Line Interface for TrustForge. Manage schemas, generate cryptographic proofs, configure policies, and control the local daemon.

## Install

```sh
bun add @trustforge-protocol/cli
# or
npm install @trustforge-protocol/cli
```

## Usage

```sh
bunx @trustforge-protocol/cli --help
```
