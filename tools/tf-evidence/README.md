# @trustforge-protocol/evidence

TrustForge evidence archive manager. Record, query, and cryptographically verify historical actions and capability delegations.

## Install

```sh
bun add @trustforge-protocol/evidence
# or
npm install @trustforge-protocol/evidence
```

## Usage

```sh
bun run src/cli.ts --help   # assemble and verify evidence bundles
```
