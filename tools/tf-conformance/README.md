# @trustforge-protocol/conformance

TrustForge protocol conformance test suite. Ensures all language implementations and adapters strictly adhere to the TrustForge specification.

## Install

```sh
bun add @trustforge-protocol/conformance
# or
npm install @trustforge-protocol/conformance
```

## Usage

```ts
import * as pkg from "@trustforge-protocol/conformance";
```
