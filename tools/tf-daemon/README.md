# @trustforge-protocol/daemon

The TrustForge background daemon. Handles key management, packet routing, capability caching, and secure session orchestration.

## Install

```sh
bun add @trustforge-protocol/daemon
# or
npm install @trustforge-protocol/daemon
```

## Usage

```sh
bun run src/cli.ts start   # local enforcement daemon (decide/proof endpoints)
```
