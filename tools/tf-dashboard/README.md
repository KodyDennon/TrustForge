# @trustforge-protocol/dashboard

TrustForge web dashboard. A visual interface for monitoring active sessions, auditing the proof ledger, and managing cryptographic policies.

## Install

```sh
bun add @trustforge-protocol/dashboard
# or
npm install @trustforge-protocol/dashboard
```

## Usage

```ts
import * as pkg from "@trustforge-protocol/dashboard";
```
