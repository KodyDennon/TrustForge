# @trustforge-protocol/types

Core semantic types, traits, and schemas powering the TrustForge protocol.

## Install

```sh
bun add @trustforge-protocol/types
# or
npm install @trustforge-protocol/types
```

## Usage

```ts
import * as pkg from "@trustforge-protocol/types";
```
