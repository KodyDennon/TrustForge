# TrustForge

**TrustForge** is an open-source trust fabric for AI-native software, secure devices, authenticated live systems, site-to-site communication, service-to-service communication, and verifiable action.

TrustForge is not just authentication.

It is a protocol stack for answering:

> Who or what is acting, under what authority, through what session, with what permissions, under what policy, across what transport, and with what verifiable proof?

## Core thesis

The next era of security is not just login.

The next era is **verifiable action by cryptographic actors over authenticated channels**.

## Major components

- TrustForge Core
- ProofWire transport
- ProofRPC
- Agent Contract
- Proof Ledger
- Compatibility Bridges
- TrustForge Daemon
- TrustForge CLI
- TrustForge Dashboard
- TrustForge Plugin System
- TrustForge Conformance Suite
